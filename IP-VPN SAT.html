CONNECTIVITY IP-VPN VSAT IP-VPN VSAT adalah layanan komunikasi satelit yang terintegrasi dengan jaringan berbasis IP sehingga menjamin komunikasi private bebas hambatan

Spesifikasi dan Fitur

  • Menggunakan perangkat satelit Very Small Aperture Terminal ( VSAT )
  • Menggunakan frekuensi satelit C Band atau Ku Band
  • Mempunyai pilihan menggunakan perangkat akselerator atau tanpa perangkat akselerator
  • Mempunyai pilihan Committed Information Rate ( CIR ) 1:1, 1:2, dan 1:4
  • Pilihan rasio uplink / downlink 1:1, 1:2 dan 1:4
